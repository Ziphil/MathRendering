Rendering is tested only on the latest desktop version of Google Chrome, so the layout may be off when viewed with a different browser.

Some practical example

The quadratic equation a x 2 + b x + c = 0 has real roots when D := b 2 − 4 a c ≥ 0 holds, and those roots are explicitly written as

x = − b ± √ b 2 − 4 a c 2 a

Otherwise D < 0 , in which case the equation does not have real roots but two distinct imaginary roots.
